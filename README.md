# CRM Project

Personal project for building a CRM (Customer Relationship Management) application, developed for study and experimentation purposes.

Tech Stack

* Angular 21.2.6
* TypeScript
* RxJS

Prerequisites

* Node.js (recommended LTS version)
* Angular CLI

Install Angular CLI if not already installed:

npm install -g @angular/cli

Installation

Clone the repository and install dependencies:

git clone <your-repo-url>
cd <project-folder>
npm install

Run the project

Start the development server:

ng serve --open

The application will be available at:

http://localhost:4200/

Purpose of the Project

This project is intended to:

* Practice Angular development
* Explore CRM-related features and patterns
* Experiment with frontend architecture and state management
